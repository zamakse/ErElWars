# 에르엘워즈1 (AREL WARS 1) — 원작 레퍼런스 기사

> 출처: 게임빌 공식 보도자료 / 게임 소개 기사 (에르엘워즈1 출시 당시)
> 정리 목적: ZALIWA 개발 시 원작 시스템 이해 참고용

---

## 장르 정의 — 진격형 디펜스

- 기존 디펜스 게임의 **배치형·방어 위주** 방식을 탈피
- **유닛 진격을 통한 공격·수비 조합**이 핵심 — 공격과 방어 모두 전략적 판단 필요
- 높은 자유도 + 액션성 + 타격감을 갖춘 "한국형 디펜스"로 평가받음
- 스마트폰 터치에 최적화된 조작 구조

---

## 세계관 및 종족

- **3종족 전쟁** 배경: 흄(Hume), 버스터(Buster), 엘븐(Elven)
- 유저는 세 종족 중 하나의 **영웅을 선택**해 플레이
- 동화적 그래픽과 캐릭터 디자인으로 스마트폰 최적화

> **참고**: 에르엘워즈2에서는 흄+엘븐이 '유니언'으로 통합, 데븐 종족이 추가됨

---

## 유닛 구성

| 구분 | 내용 |
|------|------|
| 총 유닛 수 | 54개 |
| 유닛 등급 | 영웅(Hero) / 유니크(Unique) / 일반(Normal) |
| 병종 | 보병, 기병, 공군(비행), 포병, 의무병, 영웅 |
| 육성 방법 | 경험치 획득을 통한 레벨업 |
| 합성 시스템 | 유닛 조합으로 새로운 유닛 생성 가능 |

---

## 핵심 시스템

### 유닛 상성
- 포병, 보병, 기병, 공군 등 병종별 상성 관계 존재
- 적절한 시점·공간에 유닛을 배치/진격시키는 전략이 핵심

### 영웅 시스템
- 영웅 캐릭터 전장 직접 출진
- 피해가 크면 **본진 소환 → 치료 → 재출진** 가능 (전략적 운영)
- RPG 요소와 절묘하게 결합

### 육성·강화
- 유닛 아이템 장착으로 강화 가능
- 유닛 조합으로 새로운 유닛 생성
- RPG적 육성 요소가 디펜스에 녹아들어 있음

### PvP (대전 모드)
- 유저가 직접 설계·구성한 부대로 대전
- **서버에 저장된 타 유저의 부대를 다운받아 전투** (비동기 PvP)
- 각기 다른 전략과 부대 편성이 재미 요소

---

## 스테이지 구성

- 8개 전투 지역
- 각 지역당 다수의 전장 → 총 120여 개 전장
- 노드 기반 순차 해금 방식으로 추정

---

## 부대 편성 옵션

- 유닛 편성 + 업그레이드 + 출진 부대 선택
- 운영 콘셉트 예시:
  - 공중부대 중심
  - 보병 + 포병 혼합
  - 기병 돌격형
  - 의무병 포함 지속형
